# Weather-App
# I had developed a real-time weather forecasting app project to forecast the current weather in any city.
# Integrated with openweathermap API to retrive live weather data including temperature, weather descriptions, humidity and wind speed.
# Designed and implemented Django models, views, and forms to handle user input, API requests, and dynamic rendering of results through Django templates.

# Technologies used
# Frontend development: HTML, CSS
# Backend development: Python, Django
# Database: SQLite
